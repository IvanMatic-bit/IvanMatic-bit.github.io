Ticket System / Škola

GitHub repostory

Za izradu ovog projekta koristio sam tehnologije PHP, HTML, CSS, JavaScript, jQuery, MySQL

Ovaj program je namijenjen za kreiranje i evidenciju zahtjeva. U ovom primjeru namijenjen je za školu. Imamo 3 vrste korisnika (Administrator, Obični korisnik i korisnici sa ulogama). Administratori vrše evidenciju korisničkih računa i kreiraju nove. Obični korisnici mogu da šaju nove zahtjeve. Zahtjev sadrži naslov, sadržaj i potrebno je odabrati za koga je namijenjen. U bazu se još spremaju i ID korisnika koji je kreirao zahjtev i vrijeme kreiranja. Korisnici sa ulogama izvršavaju zadatake. Uloge su (IT support, domar i higijeničar). Nakon prijave korisnici sa ulogama imaju pregled zahtjeva namjenjenih za njihovu ulogu. Nakon izvršavanja zahtjev se prebacuje u tabelu arhiva, gdje se pored podataka o zahtjevu još dodaje i datum i vrijeme izvršavanja i ID korisnika koji ga je izvršio.


Blue game / Igrica

GitHub repostory Započni igricu

Za izradu ovog projekta koristio sam HTML, CSS, JavaScript i jQuery

Pravila igre:

Igrač mišem kontroliše svemirski brod koji putuje kroz. Izbjegava meteore i skuplja hranu. Hrana donosi +20 poena, ako se pokupi u prvih 5 sekundi. Svako pet sekundi broj poena koje se može pokupiti se smanjuje za 5. Minimalno donosi 5 poena. Svemirski brod poraste svaki put kada se pokupi hrana. Kraj igre je kada svemirski brod dotakne meteor.


Videoteka

GitHub repostory

Za izradu ovog projekta koristio sam tehnologije PHP, HTML, CSS, JavaScript, jQuery, MySQL

Program za iznajmljivanje i kupovinu filmova i serija. Korisnik kreira novi korisnički račun i prijavljuje se na sistem. Nakon prijave na sistem korisnik ima opciju pregleda i pretraživanja filmova i serija. Na ovom projektu je potrebno implementirati da korisnici/administratori imaju mogućnost dodavanja novih filmova i serija. U tabeli korisnik polje UserAdmin označava da li je korisnik administrator. Ukoliko jeste potrebno je dodati funkcionalnost dodavanja novih filmova i serija.
